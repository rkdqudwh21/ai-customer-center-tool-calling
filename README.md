# 📞 AI 고객센터 (Tool Calling 실습)

## 📌 프로젝트 개요
LangChain Tool Calling 구조를 이해하기 위한 실습 프로젝트입니다.  
고객 문의를 입력하면 상황에 맞는 Tool을 호출하여 문제를 해결하는 흐름을 구현했습니다.

---

## 🛠️ 사용 기술
- Python
- Streamlit
- LangChain
- OpenAI API

---

## ⚙️ 주요 기능
- 고객 정보 조회 (CRM)
- 환불 정책 조회
- 배송 상태 확인
- 환불 처리
- 상담 티켓 생성

---

## 🔄 핵심 구조
사용자 입력 → LLM → TOOL 선택 → 실행 → 결과 반환

---

## ✨ 내가 수정한 부분
- 고객 데이터 추가 (C-3003)
- 정책 케이스 확장 (교환, 고장)
- 주문번호별 배송 상태 분기 추가
- 사용자 입력에 따라 서로 다른 Tool 호출 흐름 확인

---

## 💡 배운 점
- LLM이 외부 기능을 호출하는 Tool Calling 구조 이해
- 단순 챗봇이 아닌 Agent 개념 경험
